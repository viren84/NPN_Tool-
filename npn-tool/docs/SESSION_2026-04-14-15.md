## Features Shipped

### 1. PDF Export System (`2f987a6`)
- Added `pdf-lib` dependency
- Editable PDF generation with form fields (197 fields in combined PDF)
- Single-doc PDF via `?docType=` query param
- Combined PDF with cover page + TOC
- Greek letter mapping, markdown fence stripping, smart table columns
- XSS fix on `/api/search` query parameter

### 2. Document Review UX (`2f987a6`, `db6f86f`)
- 4 button toolbar: Discard, Import, Save, Export PDF, Regenerate, Approve
- Fixed stale content bug when switching between draft documents
- View, Download, Delete buttons on each product document
- Per-document PDF download icons in DocumentsClient, ApplicationEditor, PackageClient

### 3. Product Pipeline Foundation (`4b88ce1`, `5f11755`, `4a6959f`)
- New `Product` model with 15-stage lifecycle (research → withdrawn)
- CRUD API at `/api/products`
- Pipeline UI with stage filter pills, product cards, create modal
- Detail page with stage timeline stepper
- "Product Pipeline" navigation added to sidebar

### 4. Dr. Naresh Review Workflow (`4b88ce1`, `4a6959f`)
- Review fields on Product model (reviewStatus, reviewerId, timestamps)
- Review request/decision API at `/api/products/[id]/review`
- Reviewer picker (selects team members, not self)
- Approve / Reject / Needs Changes decision flow

### 5. Secure Vault Foundation (`4b88ce1`)
- `SecureVaultItem` model with owner-only access + sharedWith JSON
- CRUD API at `/api/vault` with access enforcement
- File upload support (50MB limit), access count tracking

### 6. Tool 2 Read-Only API (`4b88ce1`)

- X-API-Key header authentication

- `CompetitorProduct` model for Amazon/website analysis

- `ProductStageData` model for dynamic per-stage sections

- Upload-to-fill: AI extracts data from uploaded PDFs
- Excel presets for reusable formulation templates
- Data source tracking

### 8. Product Detail 4-Tab Layout (`9a05f19`, `6d7a519`, `8b18b2b`)
- Overview | Documents | AI Research | Review & Notes tabs
- Documents tab with upload + extraction status + per-doc AI-extracted data
- **AI Research → Prompt Generator** (no token cost)
  - 9 prompt templates (ingredient, competitor, market, regulatory, monograph, formulation, dosage, condition, combination)
  - Copy to clipboard + Download .txt
  - Structured JSON output format for ingestion back

### 9. Health Condition Classification (`1e8e0ed`)
- `healthConditionsJson` field on ProductLicence
- 20-condition taxonomy (immune, cardiovascular, digestive, etc.)

- 53 ingredients backfilled from existing 30 synced licences
- New `/api/ingredients/[id]/usage` endpoint (shows licences/apps/products using each ingredient)
- Expandable row UI with Identity, Regulatory, Safety, Dosing, Usage, Suppliers sections
- Multi-select checkboxes for Compare (2-5 ingredients)
- Source filter (NHPID / LNHPD / Manual / CSV)
- Detail page at `/ingredients/[id]` for new-tab opens
- "Open ↗" link on each row
